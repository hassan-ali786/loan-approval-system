# Loan Approval Prediction System

## Overview

The **Loan Approval Prediction System** is an end-to-end machine learning web application that predicts whether a loan application will be **approved or rejected** based on user information.

This project combines **machine learning, data preprocessing, and a modern web interface** to provide instant predictions. It simulates a simplified decision-support tool that could be used in **banking and fintech environments**.

The system allows users to enter applicant information through an interactive web form and receive a prediction generated by a trained machine learning model.

---

## Features

* **Machine Learning Prediction**
  Uses a trained Random Forest model to predict loan approval outcomes.

* **Interactive Web Interface**
  Clean and responsive multi-step form for entering applicant details.

* **Real-Time Prediction**
  Users receive instant approval or rejection results after submitting the form.

* **Data Preprocessing Pipeline**
  Handles missing values and categorical feature encoding.

* **Professional UI Design**
  Gradient colors, responsive layout, and modern user interface components.

* **End-to-End Workflow**
  Dataset → Model Training → Model Saving → Web Application → Prediction.

---

## Dataset

This project uses a public dataset from Kaggle:

Loan Prediction Dataset
https://www.kaggle.com/datasets/altruistdelhite04/loan-prediction-problem-dataset

The dataset contains applicant information such as:

* Gender
* Marital Status
* Dependents
* Education
* Employment Status
* Applicant Income
* Co-Applicant Income
* Loan Amount
* Loan Term
* Credit History
* Property Area

These features are used to train the machine learning model.

## Installation

### 1. Clone the Repository

git clone https://github.com/hassan-ali786/Loan-Approval-Prediction.git

cd Loan-Approval-Prediction

---

### 2. Create Virtual Environment (Optional)

python -m venv venv

Windows:
venv\Scripts\activate

Linux / Mac:
source venv/bin/activate

---

### 3. Install Dependencies

pip install -r requirements.txt

---

### 4. Train the Machine Learning Model

python train_model.py

This will generate the trained model file:

model/loan_model.pkl

---

### 5. Run the Web Application

python app.py

---

### 6. Open in Browser

http://127.0.0.1:5000
